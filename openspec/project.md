# One-API-Libra 项目

## 项目概述
基于 one-api 的大模型 API 路由平台，目标是转型为类似 OpenRouter.ai 的统一 AI 模型 API 平台。

## 技术栈
- **后端**: Go + Gin Framework
- **数据库**: MySQL + GORM
- **缓存**: Redis
- **前端**: React + Material-UI
- **认证**: JWT + OAuth2

## 核心功能
- 统一的 OpenAI 兼容 API 接口
- 多渠道/多供应商模型路由
- 用户余额和额度管理
- API Key 管理
- 请求日志和统计

## 目标愿景
将项目改造为 OpenRouter 风格的平台：
- 美元余额系统
- 模型市场和定价
- 智能路由和故障转移
- 用户友好的前端界面

## 编码规范
- Go 代码遵循官方代码规范
- 使用 GORM 进行数据库操作
- API 遵循 RESTful 设计
- 前端使用 TypeScript
